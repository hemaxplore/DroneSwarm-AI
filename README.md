# 🚁 Autonomous Drone Swarm Coordination using MARL & CNN

An intelligent drone swarm system that enables multiple drones to coordinate autonomously using Multi-Agent Reinforcement Learning (MARL) and Convolutional Neural Networks (CNN). This project focuses on decentralized decision-making, real-time obstacle detection, and adaptive swarm behavior in dynamic environments.

# ✨ Features

**🚁 Multi-Drone Coordination** – Multiple drones act as intelligent agents and cooperate autonomously

**🧠 MARL-Based Learning** – Uses reward-based learning for adaptive decision-making

**🧍Obstacle Detection** – CNN model detects fire, buildings, and environmental obstacles

**🔄 Decentralized Control** – No central controller; each drone makes independent decisions

**📊 Real-Time Monitoring Dashboard** – Displays drone positions, detected objects, and system output

**⚡ Adaptive Navigation** – Handles dynamic obstacles and environmental changes

**📈 Performance Evaluation** – Tracks rewards, success rate, and collision metrics
